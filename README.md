# pico-project
Heart rate monitor project for Raspberry Pi Pico

Github repository for Metropolia school project utilizing the Raspberry Pi Pico W board with extension parts.
Project uses: SSD1306 OLED-screen, Crowtail Pulse Sensor, Raspberry Pi Pico W, Metropolia add-on extension board for the Pico W
The project uses MicroPython as it's programming language. No c++ sadly


!!! IMPORTANT !!!
To make the program work on your own device, remember to change network information and Pico MAC-address in the KubiosHRV.py file.
